# NLP Sentiment Analysis

## Objective
Classify customer/movie reviews as Positive or Negative using NLP.

## Technologies
- Python
- NLTK
- Scikit-Learn
- TF-IDF
- Naive Bayes

## Workflow

1. Text Cleaning
2. Tokenization
3. Stopword Removal
4. Lemmatization
5. TF-IDF Vectorization
6. Naive Bayes Classification
7. Sentiment Prediction

## Result

Achieved over 88% accuracy on sentiment classification.
