# Titanic Data Science Project 🚢

This project was completed as part of the DecodeLabs Data Science Internship. The objective is to understand the complete data science workflow, starting from dataset exploration to building a predictive model.

## Project Objectives

- Understand the dataset structure and features
- Perform data cleaning and preprocessing
- Analyze patterns and trends in the data
- Visualize key insights
- Build a predictive model for survival prediction

## Dataset

The project uses the Titanic dataset, which contains information about passengers aboard the Titanic, including:

- Passenger ID
- Passenger Class
- Name
- Gender
- Age
- Ticket Information
- Fare
- Cabin Details
- Embarkation Port
- Survival Status

## Tasks Completed

### Task 1: Data Collection & Dataset Understanding
- Loaded the Titanic dataset
- Identified columns and data types
- Analyzed dataset size and structure
- Understood feature significance

### Task 2: Data Cleaning & Preprocessing
- Handled missing values
- Removed duplicate records
- Corrected data formats
- Prepared data for analysis

### Task 3: Exploratory Data Analysis (EDA)
- Generated summary statistics
- Identified trends and patterns
- Detected outliers
- Derived meaningful insights

### Task 4: Data Visualization
- Created histograms
- Generated bar charts
- Built correlation heatmaps
- Visualized survival trends

### Task 5: Predictive Modeling
- Applied machine learning techniques
- Trained a survival prediction model
- Evaluated model performance
- Interpreted results

## Key Insights

- Passenger class influenced survival rates.
- Female passengers had a higher survival rate than males.
- Younger passengers showed different survival patterns.
- Fare and class were important indicators of survival.
