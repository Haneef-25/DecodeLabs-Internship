# Customer Segmentation using PCA and K-Means

## Objective
To identify hidden customer groups using unsupervised learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Steps Performed
1. Data Cleaning
2. Feature Scaling
3. Principal Component Analysis (PCA)
4. Elbow Method
5. Silhouette Score
6. K-Means Clustering
7. Customer Persona Creation

## Results
Customers were segmented into multiple groups based on spending behavior and income patterns.

## Learning Outcomes
- Dimensionality Reduction
- Unsupervised Learning
- Customer Segmentation
- Business Intelligence
